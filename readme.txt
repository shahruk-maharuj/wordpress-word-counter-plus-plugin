== Installation ==

1. Upload the `word-counter-plus` directory to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to the Word Counter Plus settings page and customize the plugin according to your preferences.
